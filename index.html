Resumen

Este trabajo estudia el comportamiento dinámico del péndulo doble acoplado, un sistema mecánico que exhibe caos determinista debido a su naturaleza no lineal. Se construyó un montaje experimental con dos varillas articuladas, y se registró su movimiento mediante videos analizados con el software Tracker para obtener trayectorias angulares.

Las ecuaciones de movimiento se derivaron utilizando el formalismo lagrangiano y se resolvieron numéricamente en Python con el método Radau. Se compararon las trayectorias experimentales y simuladas, observando sensibilidad a las condiciones iniciales y patrones caóticos.

Los resultados confirman cualitativamente la presencia de caos determinista, aunque se identificaron discrepancias debido a fricción y limitaciones experimentales. Este estudio introduce conceptos fundamentales de mecánica clásica, sistemas no lineales y caos.

Introducción

El péndulo doble es un sistema mecánico que combina simplicidad estructural con una dinámica compleja, siendo un ejemplo clásico de caos determinista. A diferencia del péndulo simple, cuyas ecuaciones son lineales bajo pequeñas oscilaciones, el péndulo doble presenta un comportamiento no lineal que produce trayectorias impredecibles para ciertas condiciones iniciales.

Este caos determinista, caracterizado por una extrema sensibilidad a las condiciones iniciales, es el foco de este trabajo. A través de experimentos y simulaciones, exploramos estos fenómenos para entender mejor los sistemas no lineales.

Metodología Experimental

Se construyó un montaje con dos varillas articuladas. El movimiento fue registrado en videos y analizado con el software Tracker para obtener las trayectorias angulares de las varillas.
