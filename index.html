Análisis Experimental y Teórico del Péndulo Doble: Caos Determinista y Mecánica Lagrangiana John Mauricio Mesa Echeverri1 † Aleph Sebastián Enciso Cruz1 ‡ 1Instituto de Física, Universidad de Antioquia UdeA, Calle 70 No. 52-21, Medellín, Colombia Resumen Este trabajo estudia el comportamiento dinámico del péndulo doble acoplado, un sistema mecánico que exhibe caos determinista debido a su naturaleza no lineal. Se construyó un montaje experimental con dos varillas articuladas, y se registró su movimiento mediante videos analizados con el software Tracker para obtener trayectorias angulares. Las ecuaciones de movimiento se derivaron utilizando el formalismo lagrangiano y se resolvieron numéricamente en Python con el método Radau. Se compararon las trayectorias experimentales y simuladas, observando sensibilidad a las condiciones iniciales y patrones caóticos. Los resultados confirman cualitativamente la presencia de caos determinista, aunque se identificaron discrepancias debido a fricción y limitaciones experimentales. Este estudio introduce conceptos fundamentales de mecánica clásica, sistemas no lineales y caos. Introducción El péndulo doble es un sistema mecánico que combina simplicidad estructural con una dinámica compleja, siendo un ejemplo clásico de caos determinista. A diferencia del péndulo simple, cuyas ecuaciones son lineales bajo pequeñas oscilaciones, el péndulo doble presenta un comportamiento no lineal que produce trayectorias impredecibles para ciertas condiciones iniciales. Este caos determinista, caracterizado por una extrema sensibilidad a las condiciones iniciales, es el foco de este trabajo. A través de experimentos y simulaciones, exploramos estos fenómenos para entender mejor los sistemas no lineales. Metodología Experimental Se construyó un montaje con dos varillas articuladas. El movimiento fue registrado en videos y analizado con el software Tracker para obtener las trayectorias angulares de las varillas. Datos procesados desde archivos Excel para graficar ángulos, velocidades y energías. Metodología Teórica Las ecuaciones de movimiento se derivaron usando el formalismo lagrangiano. Se resolvieron numéricamente en Python utilizando el método Radau de solve_ivp. Parámetros: m = 0.38 kg, l = 0.31 m, g = 9.77 m/s², b = 0.0075 (coeficiente de amortiguamiento). Resultados: Energía vs Tiempo La energía mecánica total disminuye con el tiempo debido a la fricción en el sistema experimental. Resultados: Ángulos vs Tiempo Oscilaciones amortiguadas, mostrando comportamiento caótico inicial que se estabiliza. Resultados: Espacios Fásicos Las trayectorias en el espacio fásico ilustran el decaimiento de la energía y el comportamiento caótico. Discusión: Sensibilidad a Condiciones Iniciales Pequeñas diferencias en condiciones iniciales llevan a trayectorias divergentes, confirmando caos determinista. Conclusiones El estudio confirma cualitativamente la presencia de caos determinista en el péndulo doble. Discrepancias entre experimento y simulación se atribuyen a fricción y limitaciones experimentales. Este trabajo introduce conceptos clave de mecánica no lineal y caos.
